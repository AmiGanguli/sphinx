\subsection{Detailed Description}
exceptions trie, stored in a tidy simple blob we serialize each trie node as follows\-:

int result\-\_\-offset, 0 if no output mapping B\-Y\-T\-E num\-\_\-bytes, 0 if no further valid bytes can be accepted B\-Y\-T\-E values\mbox{[}num\-\_\-bytes\mbox{]}, known accepted byte values B\-Y\-T\-E offsets\mbox{[}num\-\_\-bytes\mbox{]}, and the respective next node offsets

output mappings themselves are serialized just after the nodes, as plain old A\-S\-C\-I\-I\-Z strings
