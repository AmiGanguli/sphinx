Detailed Description

global index hash used in both non-threaded and multi-threaded modes

hash entry is a CSphIndex pointer, rwlock, and a few flags (see ServedIndex_t) rlock on entry guarantees it won't change, eg. that index pointer will stay alive wlock on entry allows to change (delete/replace) the index pointer

note that entry locks are held outside the hash and Delete() honours that by acquiring wlock on an entry first
