Detailed Description

token filter for bigram indexing

passes tokens through until an eligible pair is found then buffers and returns that pair as a blended token then returns the first token as a regular one then pops the first one and cycles again

pair (aka bigram) eligibility depends on bigram_index value "all" means that all token pairs gets indexed "first_freq" means that 1st token must be from bigram_freq_words "both_freq" means that both tokens must be from bigram_freq_words
