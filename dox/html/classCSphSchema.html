Detailed Description

plain good old schema container that actually holds and owns all the fields, columns, etc

NOTE that while this one can be used everywhere where we need a schema it might be huge (say 1000+ attributes) and expensive to copy, modify, etc! so for most of the online query work, consider CSphRsetSchema
