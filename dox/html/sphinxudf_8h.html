Enumeration Type Documentation

enum sphinx_doc_factor

ranking factors interface, v2 functions that access factor values directly in the PACKEDFACTORS() blob

faster, as no memory allocations are guaranteed, but type-punned meaning that you have to call a proper get_xxx_factor_int() or xxx_float() variant the accessor functions themselves will NOT perform any type checking or conversions or in other words, sphinx_get_field_factor_int() on a float factor like min_idf is legal, but returns "garbage" (floating value from the blob reinterpreted as an integer)
