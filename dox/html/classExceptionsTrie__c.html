Detailed Description

exceptions trie, stored in a tidy simple blob we serialize each trie node as follows:

int result_offset, 0 if no output mapping BYTE num_bytes, 0 if no further valid bytes can be accepted BYTE values[num_bytes], known accepted byte values BYTE offsets[num_bytes], and the respective next node offsets

output mappings themselves are serialized just after the nodes, as plain old ASCIIZ strings
