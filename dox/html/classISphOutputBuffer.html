Detailed Description

dynamic response buffer to remove ISphNoncopyable just add copy c-tor and operator= splitting application network packets in several sphSocketSend()s causes Nagle's algorithm to wait delayead ACKs (google for it if you want) That's why we use relocation here instead of static-sized buffer. To be twice sure we're switching off Nagle's algorithm in all sockets.
